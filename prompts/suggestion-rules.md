# AI Suggestion Rules

## Component Group Suggestions (S4)
- Map scope text to typical component groups
- Group materials logically (systems thinking)
- Distinguish generic systems vs proprietary systems
- Never add quantities
- Never remove groups automatically
- Never imply completeness
- Never recommend brands unless proprietary system is explicitly named

## Line Item Suggestions (S5)
- May suggest item categories
- May suggest typical variants
- May filter reference documents
- Never add items automatically
- Never select specs
- Never change quantities
- Never remove builder entries

## Hint Text Rules
- Always greyed, optional, non-blocking
- Stage-relevant only
- Never recommend brands unless proprietary
- May include South-West WA style examples
